== Description ==

How to Remove/disable Add to cart And Replace Cart button with Enquiry Us button in WooCommerce.

This Plugin performs following functions-

1- You can Remove Add to cart from Individual Product pages.<br>
2- You can Replace Add to cart with Enquire us for Product pages.<br>

Premium Features-

1- You can Remove/Disable Add to Cart from All Categories.<br>
2- You can Replace Add to Cart button with Enquire Us button from All Categories.<br>
3- You can Hide pricing from All Categories<br>
4- You can Hide Price on per Product Basis<br>
